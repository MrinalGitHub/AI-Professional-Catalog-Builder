## Idea 1: Industrial Catalogue — Clean Automotive Spec Sheet

**Design Movement:** Swiss/International Typographic Style meets automotive catalogue design (think Ducati or Royal Enfield spec sheets)

**Core Principles:**
- Precision-first layout with strong grid alignment
- Data-dense but highly readable tables
- Monochromatic base with orange and green brand accents only
- Product images as hero elements with generous whitespace

**Color Philosophy:**
- Background: Clean white (#FFFFFF) with very light warm grey (#F8F7F5) alternating sections
- Primary accent: Orange (#FF4500) for Living Vitals branding and key highlights
- Secondary accent: Dark green (#1B5E20) for RajPraNim Enterprises
- Text: Charcoal (#1A1A1A) for body, slate (#374151) for secondary
- Table headers: Deep charcoal with orange left-border accents

**Layout Paradigm:**
- Full-width hero sections per model with product image on left, model name on right
- Specs in a two-column key-value grid (not a traditional table)
- Battery options in a card-based horizontal layout with price prominently displayed
- Sticky header with logo, scrolling single-page catalogue

**Signature Elements:**
- Orange accent lines/dividers separating sections
- Spec values in a bold monospace-style font for that "data sheet" feel
- Price tags with orange background badges

**Interaction Philosophy:** Smooth scroll between model sections, subtle fade-in on scroll

**Animation:** Minimal — fade-in on section entry, no distracting motion

**Typography System:**
- Headings: Oswald (bold, condensed — automotive feel)
- Body/Specs: Source Sans 3 (clean, readable)
- Prices/Numbers: JetBrains Mono or similar monospace

## Idea 3: Bold Catalogue — Magazine-Style Product Pages

**Design Movement:** Editorial/magazine catalogue design (like a Bajaj or Honda printed catalogue digitized)

**Core Principles:**
- Each model gets a full "page" feel with clear visual hierarchy
- Bold typography with strong contrast
- Asymmetric layouts mixing image and data
- Print-catalogue feel adapted for web

**Layout Paradigm:**
- Asymmetric two-panel layout: large product image taking 55% width, specs panel 45%
- Battery options below in a clean comparison table with alternating row colors
- Each model section separated by a bold colored divider band
- Navigation dots on the side for jumping between models

**Color Philosophy:**
- Background: White (#FFFFFF) with light grey (#F3F4F6) spec panels
- Primary: Warm orange (#E65100) for Living Vitals
- Secondary: Forest green (#2E7D32) for RajPraNim
- Accent: Electric blue (#1565C0) for Ryoto Electrix mentions
- Table rows: Alternating white and very light orange tint

**Signature Elements:**
- Diagonal cut dividers between model sections (like the Ryoto catalogue triangular design)
- Bold model names in oversized typography
- Orange pill badges for "Lithium Ion" and grey badges for "Lead Acid"

**Interaction Philosophy:** Page-snap scrolling for each model, smooth transitions

**Animation:** Slide-in from sides for image and specs, snap scroll between sections

**Typography System:**
- Headings: Bebas Neue (bold, impactful display)
- Body: Nunito Sans (friendly, readable)

## Selected Approach: Idea 3 — Bold Catalogue (Magazine-Style Product Pages)

This approach best matches the original Ryoto catalogue's style (which uses bold diagonal cuts, large product images, and spec tables). It will feel like a proper product catalogue while being digital-first. The asymmetric layout, diagonal dividers, and bold typography create visual impact while keeping the data highly readable.

**Key decisions:**
- Light background (white) so the product images stand out naturally
- Orange for Living Vitals, Forest Green for RajPraNim — consistent everywhere
- Electric blue for Ryoto Electrix references (matches their brand)
- Diagonal section dividers inspired by the original Ryoto catalogue design
- Each model gets a full-page feel with snap scrolling
